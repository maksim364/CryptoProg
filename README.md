# Практическое задание 4 — Использование криптографических библиотек

## Реализовано:
1. **Программа хеширования** (SHA-256)  
   - Читает файл, выводит хеш в hex-формате

2. **Программа шифрования/дешифрования** (AES-256-CBC)  
   - Режим: CBC  
   - Ключ генерируется из пароля через PBKDF2  
   - IV сохраняется в начале зашифрованного файла
